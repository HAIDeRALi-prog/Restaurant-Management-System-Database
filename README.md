Restaurant Management System Database

A comprehensive Restaurant Management System database developed using Oracle SQL. The project demonstrates database design principles, normalization, schema implementation, data population, querying, views, and triggers.

Features

- Normalized relational database schema
- Branch, Employee, Chef, Waiter, Supplier, Customer management
- Menu and Order management
- Payment processing records
- Data integrity through constraints
- JOIN, Aggregate, and Subquery operations
- Sales summary view
- Trigger for automatic order total calculation

Database Components

- DDL Scripts (CREATE, DROP, Constraints)
- DML Scripts (Sample Data Population)
- SQL Queries
- Views
- Triggers

Technologies

- Oracle SQL
- Relational Database Management Systems (RDBMS)
- Database Normalization
- SQL Query Optimization

Learning Outcomes

- Database Modeling
- SQL Development
- Constraint Management
- Query Writing and Optimization
- Trigger and View Implementation
